About me

I am a final-year student pursuing a Bachelor of Design (Digital Games) with Honours at Universiti Pendidikan Sultan Idris (UPSI). While my core passion lies in 3D game development—specifically utilizing the Unity engine and C# scripting to engineer interactive, local co-op experiences—I am also a versatile multidisciplinary designer.

Beyond gameplay programming and my strong aesthetic preference for low-poly art styles, my skill set spans across graphic design, UI/UX development, and cinematic video production. I thrive on exploring engaging mechanics, such as implementing single-monitor split-screen setups, while actively leveraging modern tools and AI-driven workflows to bring unique characters, environments, and compelling visual stories to life.

Projects

Dual Horizon(ONGOING)

Dual Horizon is an ongoing third-person survival shooter developed in Unity, featuring a unique local co-op experience using a dual-monitor split-screen system, where each player has their own dedicated display instead of sharing a single screen. The project combines cooperative gameplay, zombie survival mechanics, exploration, and resource management while showcasing my skills in gameplay programming, multiplayer system development, UI implementation, AI behaviour, and performance optimization using C#.

UI Project (MAIN MENU)

The Main Menu serves as the primary interface that allows players to navigate the game before entering gameplay. It is designed with a simple and user-friendly layout to ensure that players can easily access the available options. The interface includes essential functions such as starting a new game, continuing a previous game, adjusting game settings, viewing the development credits, and exiting the game.
